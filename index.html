How This Project Was Made

Concept

This project demonstrates probability theory (where each dice face has a theoretical 16.67% chance) compared against real experimental results, utilizing a simple AI prediction model to forecast outcomes.

AI Prediction Logic

For the first 5 rolls, the AI predicts randomly since there isn't enough data. After that, it predicts based on which face has appeared most frequently so far, simulating how simple AI systems learn patterns from data over time.

Fairness & Bias Explanation

According to the Law of Large Numbers, the more rolls collected, the closer each face's actual percentage should get to the theoretical 16.67%. If a face's percentage stays significantly higher or lower than expected even after many rolls, it may indicate the dice (or random number generator) is biased or unfair. This is the same principle used to detect rigged dice in real life or test random number generators.
